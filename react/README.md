# webpack-6to5-react-example

Boilerplate with React 12.2 using webpack as a module system and build tool.
Includes hot module reloading (HMR) for react components and stylesheets.


## Install

```sh
$ npm install
```

## Dev

```sh
$ npm run dev
$ open http://localhost:8080/
```

## Production

```sh
$ npm run build
```
